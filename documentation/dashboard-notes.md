# Windows Authentication Monitoring Dashboard

## Overview
This dashboard monitors Windows authentication events in Splunk Enterprise using Windows Security Event Logs.

## Event IDs Monitored
- Event ID 4624 – Successful Logon
- Event ID 4625 – Failed Logon

## Dashboard Components
1. Failed Login Counter
2. Successful Login Counter
3. Failed Logins Over Time
4. Successful Logins Over Time
5. Failed Logins by User
6. Recent Authentication Events Table

## Detection Capabilities
- Failed login detection
- Authentication monitoring
- Identification of repeated failed login attempts
- Investigation of targeted user accounts
- Visualization of authentication trends
